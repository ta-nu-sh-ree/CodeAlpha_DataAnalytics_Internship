# CodeAlpha Data Analytics Internship

## Consumer Insights through Data Analysis and Sentiment Mining

Analysis of 393,914 Amazon Fine Food Reviews (1999–2012) across three modules.

---

## Project Structure
- `notebooks/01_EDA.ipynb` — Exploratory Data Analysis
- `notebooks/02_Visualization.ipynb` — Data Visualization
- `notebooks/03_SentimentAnalysis.ipynb` — Sentiment Analysis

## Key Findings
- 63.7% of reviews are 5-star rated
- Average rating: 4.18 / 5
- 87.4% of reviews carry positive sentiment (VADER)
- Sentiment score strongly correlates with star rating
- Critical (1-2 star) reviews get marked more helpful by readers

## Tools Used
Python, Pandas, NumPy, Matplotlib, Seaborn, Plotly, NLTK (VADER), WordCloud

## Dataset
Amazon Fine Food Reviews — 568k reviews, cleaned to 393k
